Contribute

We encourage everyone to make contribution to the project! If you want to make ExtWebDriver better with new features, enhancements or bug fixes, here's how you can contribute.

  • Create a GitHub account
  • Submit a ticket in GitHub project (please verify the ticket doesn't already exist by searching here)
  • Fork the repository in GitHub
  • Code code code!
  • Create unit tests
    • For every class you are writing/modifying, you must create/modify unit tests
    • All unit tests run against a locally deployed app (static HTML files under 'src/test/resources/webapp')
    • You may extend an existing HTML to test your code or create a new HTML
    • You should not break existing tests
  • Build
    • mvn package
      • Compile code
      • Create package jar
    • mvn verify
      • Compile
      • Deploy local app on Jetty server (default port is 9090)
      • Run integration tests (unit tests)
  • Make a pull request
    • Include reference to the ticket
    • After review, the pull request will be merged to the master

When you make a pull request, you must sign off your work. You can automatically sign off by using "git commit -s" command. When you sign off, you are making an agreement to the following:

Report Bugs

When you encounter bugs, please try to find if the issue already has been reported by searching here. Once you identify your issue is new, please create a new ticket by clicking 'New Issue'. Please make sure you provide a clear description of the issue along with a testable scenario.

If you want to get involved in fixing bugs, first read the steps outlined above under 'Contribute'. Then, you can assign yourself to the bug as the owner and start developing your code. Once you are complete with your code, tested, built, please make a pull request for review. Once the review is complete, your code fix will be merged into master.

Communicate

If you want to get involved, you can start by contributing to the GitHub project or reach out to opensource@finra.org.

If you have any questions or discussion topics, please post them on Google Groups
